记录一下我的第一个vivado工程
环境：Vivado2019.2
参考：https://blog.csdn.net/qq_41154156/article/details/80989125

新建工程

1.打开Vivado - Create Project-File - New Project新建工程

2.出现一个设置向导，点击Next，修改工程名, 修改存放路径，一路点击next

3.为你的项目选择一个器件，这里如果不知道选哪个可以随意选择，之后可以更改。选择好之后点击Next.

添加源文件

1.工程界面如下，左边的PROJECT MANAGER（工程管理）下面有一些导航栏例如Settings（设置）、Add Sources（添加源文件）等，中间Sources（源文件），右边是Project Summary（项目摘要）

2.点击Add Sources（添加源文件），选择Add or create design sources添加设计源文件，点击Next

3.选择 Create File,在弹出的页面中输入文件名multi_vote，点击OK.

4.弹出模块定义的窗口，在这里可以定义模块名。定义输入输出，定义a,b,c 为输入，f 为输出。点击OK

5.在Design sources下面多了一个.v的设计源文件，双击打开如下 (源文件编辑窗口的字体比较小，如果想修改字体，调整方法：Tools->settings->Text Edior->Fonts and Colors可以更换代码字体、调整大小等等)

6.编写代码。按ctrl+s键保存代码。

7.查看RTL分析，看它的电路是什么样的。点击左边栏RTL ANALYSIS下面的Open Elaborated Design.得到下面的电路图

仿真和编写约束文件

1.点击Add Sources（添加源文件），选择Add or create simulation sources添加设计源文件，点击Next.在出现的页面点击Create File，填写文件名：sim_multi_vote，点击OK，然后点击Finish

2.弹出仿真文件的信息，直接点击OK，在弹出的窗口点击yes

3.找到Simulation Sources 下面的仿真文件sim_multi_vote,双击打开，添加代码。测试文件编写完成后按ctrl+s保存文件，同时vivado会自行进行代码的检验，检验报告在下方的Messages，如果有错误也会在哪显示出来

4.运行仿真。点击左侧的Run Simulation 选择Run Behavioral Simulation ,查看仿真结果

可以看出，波形能够符合三人表决电路结果
