接着上一篇发送hello world，记录一下zynq以太网项目实现过程
环境：Vivado2020.2，Vitis，Miz701N
参考：https://blog.csdn.net/FPGADesigner/article/details/88751502

实现结果

本次实验实现了PS端通过UDP接收PC端传输过来的数据，并通过BRAM保存在指定内存区域中。
缺陷：由于还未能查询到内存擦除指令，暂时只能保存一次传输过来的数据，之后数据再不保存，只能通过重新Debug的方式再保存数据。

PL端设计

1.添加CPU

2.勾选GP端口

2.使能网口及串口


3.系统频率设置及FCLK频率设置，我的板子用的是50MHz，FCLK设置为100MHz

4.下载设置，看个人板子要求

5.添加两个BRAM，并设置Number of BRAM interfaces为1，其它默认


6.添加Block Memory Generator, 并设置Memory type为True Dual Port RAM


7.设置完成后执行Run Block Automation和Run Connection Automation，完成连线后如图

注意：使用Vavido高版本时可能会产生警告，类似如下图

解决方法：打开vivado的address editor，你可能会发现有一个Unassigned，如图1，右键assign，并重新设置地址即可
